Примеры работ

  • 1. Реализовать галерею

    Требования:

    1. Использовать разные изображения для превьюшек и большого изображения (в этом и заключается цимес)
    2. Автоматически листать на следующее изображение через 5 секунд
    3. Не листать галерею после того, как пользователь кликнул на одну из превьюшек
    4. Количество галерей на странице может быть любым
    5. Количество изображений в галерее может быть любым
    6. Галерею можно листать с клавиатуры. Но только одну из всех галерей на странице. Листается та, с которой пользователь работал в последний раз. Или над которой находится мышка в данный момент.
    7. Должно работать в 8 ие.

    Реализация: Галерея  Код: Css  JavaScript

  • 2. Реализовать многоуровневое контекстное меню как в Windows. Открываться должно по правой кнопке в любой части страницы.

  • 4. Реализовать инпут-пароль, в момент ввода проверять сложность вводимого пароля. Сложность градировать по 5-бальной шкале. Показывать уровень сложности пароля под полем ввода. Детали реализации функции проверки сложности пароля не интересны (можно считать количество символов, например).

    Написать валидатор, который будет проверять верность вводимого email-а. Каждую ошибку сопровождать соответствующим сообщением. Возможные ошибки: не заполненное поле, не валидный email, уже занятый email. Предполагаем, что список занятых email-лов есть, и он хранится в глобальной переменной.

    Реализация: Email и Password валидация  Код: Css  JavaScript
